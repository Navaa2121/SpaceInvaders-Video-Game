# SpaceInvaders-Video-Game

This project was completed as part of an Object-Oriented Programming course. It focuses on improving and extending an existing Space Invaders style video game.

## My Contributions
  - Debugged and fixed syntax and compile errors
  - Applied OOP principles (Single Responsibility Principle and Open/Closed Principle)
  - Created a Menu system allowing players to select different characters, bullet types, and       including an option for custom images via local images or URL input
  - Implemented a continuous shooting mechanic for improved gameplay responses
  - Added an animated starfield background to enhance the visual experience of the video game

## Technologies
  - Java
  - VS Code
  - GitKraken
  - Bitbucket

## Note
This project demonstrates experience working within an existing codebase and applying software engineering principles to extend and enhance functionality.
